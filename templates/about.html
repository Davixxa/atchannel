Why create @channel?

Why not? Steins;Gate is one of my most favorite animes and I was disappointed to see that @channel did not actually exist (though it is technically based off 2chan, and supposedly operates like 4chan.) I just thought it would be nice to have exist. I can also show my friends, family, and future employers "Hey, I made this. Isn't that a neat thing."

How did you make @channel?

@channel was originally coded entirelly in javascript using Node.js and MongoDB on with a framework called Meteor. However, meteor was too slow for retrieving posts, so I had to dump it and now the site uses Flask instead.

Can I contribute to @channel?

YES PLZ!!! I would love for people to contribute to making @channel a real thing. The source code for @channel is available on Github if you want to help develop it. You can also help by pointing any out small mistakes like bugs spelling errors you find here on SuggestionsChannel. There is also a list of things you can help out with below.
