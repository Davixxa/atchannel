What is @channel?

The goal of this website is to replicate the @channel from the anime Steins;Gate as much as possible (making this basically a parody of 4chan). Here you will find various channels dedicated to a variety of topics created by users. Users do not need to register an account before participating in a channel.

Before you post anything to @channel, be sure to check out the rules regarding posts. You can find more about @channel (features, development, todos, etc.) here.

Check out the main channel here or find a channel that interests you here.

Quick tips about using @channel

Switching between the anime and VN design

A lot of people commented that the design of @channel did not look like how it was portrayed in the visual novel. You can alternate between both styles in the settings (the little gear in the upper right corner.)

Turning on and off the scrolling animation

When on a channel, resizing each post while scrolling (as it was portrayed on the anime) took up a lot of CPU. You can choose to turn off and on these animations to make it easier on your computer in the settings. If you are on the VN style or viewing from a mobile devices, animations are automatically disabled.

@channel uses markdown

Why? Because markdown is awesome! Don't know markdown? That's fine. You don't need to know it to submit posts. If you want to use markdown, here's a little cheatsheet.

Helping build @channel

If you want to contribute to developing @channel, the code is open source on Github. If you want to help but don't know how to code, you can check out some of the things on my todo list for improving @channel. If you also spot some small things like bugs or spelling errors, post them on SuggestionsChannel.
