Tips for creating a new channel

  • Please familiarize yourself with the rules before posting. Posts that go against the rules will be removed.
  • Markdown is enabled for posts, so feel free to be as creative as you like with your post. (Markdown cheatsheet)
  • You can tag other posts by including the ID of that post between < and >. For example, if you want to reply to or tag the post with the ID "5510d204485f6c135f8f664a", just include <5510d204485f6c135f8f664a> anywhere in the post and it will be added to the list of tags. You can do this for multiple posts in the same message.
  • Posting HTML will not work. For example, posting <strong>Bold text</strong> will not result in Bold text. Instead use **Bold text**.
